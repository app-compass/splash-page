What is AppCompass?

AppCompass was created by Plus 3 Interactive, a small development shop. To save our time and serve our diverse clients better, we needed one core system to build and manage complex web applications and multiple websites.

So we created AppCompass, an open-source Laraval framework for building and managing applications – and associated websites - that use SQL databases.

The core components:

Modular

CLI package loading & management module.

FormBuilder

Data structure management for building forms.

ResourceBuilder

API structure management to establish end-points.

PilotCP

Front-end control panel for visual control over data.

And more...

Other components we’ve built for AppCompass include Slingshot for auto deployment of apps, sites, etc; our modern full-featured PilotCMS; geographic information management with Geo; and for convenience, Blogger.

AppCompass is designed for Laravel developers, development shops, and companies managing multiple websites with complex web applications.
